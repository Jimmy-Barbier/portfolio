PROJET PERSONNEL · PUBLIÉ SUR GOOGLE PLAY

CloudPass FR — Préparation CLF-C02

Application mobile · React Native · Android

Voir sur Play Store →

Pourquoi ? En préparant le CLF-C02, aucune application de révision en français n'offrait un niveau de qualité satisfaisant. J'ai donc construit la mienne — pour moi d'abord, puis publiée pour la communauté francophone AWS.

Comment ? Développée en React Native avec Expo, l'application couvre l'intégralité du programme CLF-C02 : 13 modules de cours structurés par domaine AWS, 30 flashcards pour ancrer les concepts clés, 3 examens blancs chronométrés avec correction détaillée, statistiques de progression par domaine et mode révision ciblée sur les points faibles. Interface 100% en français. Build Android via EAS, publication complète sur Google Play Console — de la conception au store, mené en autonomie totale.

React Native Expo EAS Build Google Play AWS CLF-C02 JavaScript
